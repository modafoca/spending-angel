# Spending Angel

A tiny Chrome extension that's the angel on your shoulder when you're about to spend.

You tell it once what you're saving for. Then, when you land on a shopping site or click "Buy now," it slides in from the corner with a cartoon angel and a sound, and reminds you: *"Hey. Stop. You're saving for Tokyo trip."*

That's it. No tracking, no math, no dashboards. The joke does the work.

(README expanded in a later commit — see the bottom of this file for install steps.)

## Install (dev mode)

1. Clone this repo.
2. Open `chrome://extensions`.
3. Toggle **Developer mode** on (top-right).
4. Click **Load unpacked** → pick this folder.
