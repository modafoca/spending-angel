# sounds/

The four `.mp3` files in this folder are **silent 0.5-second placeholders** generated with ffmpeg, so the extension loads without 404s while you decide on real audio. Chrome will play them — you just won't hear anything.

## What to record / drop in

| File | Vibe | Suggested length |
|---|---|---|
| `stop.mp3` | The Classic — "Hey. Stop. Don't do that." Ian's voice or stand-in. | ~2s |
| `bonk.mp3` | Cartoon BONK / mallet hit. Pairs with the visual wobble. | ~0.5s |
| `mom-sigh.mp3` | Long, theatrical disappointed sigh. No words. | ~1.5s |
| `wah-wah.mp3` | The classic sad trombone. | ~1.5s |

Just keep the filenames. To add more options, drop a new `.mp3` here and add a matching `<option>` in `popup.html`.
